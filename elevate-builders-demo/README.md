# Elevate Builders — MUSTAQEEM ERP Demo

A responsive, clickable front-end demonstration of the proposed MUSTAQEEM Construction ERP for Elevate Builders. It uses sample data only and does not connect to the production Oracle APEX application or database.

## Preview locally

Open `index.html` in Chrome, Edge, Safari or Firefox. The demo credentials are pre-filled; select **Enter demo**.

## Publish with GitHub Pages

1. Create a new **private or public** GitHub repository.
2. Upload everything inside this folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/ (root)`, then select **Save**.
6. GitHub will show the public demo URL after deployment.

## Scope represented

- Executive dashboard and portfolio KPIs
- Projects and tender pipeline
- Procurement and supplier controls
- Inventory and reusable assets
- Workforce, attendance and payroll indicators
- Finance, approvals and management reports

## Important

This is a presentation prototype. It contains no database, authentication, transaction processing or confidential client information. Replace the sample figures before any formal presentation if needed.
